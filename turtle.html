UI:

  • arrow keys move the turtle
  • space toggles drawing
  • 'ESC' clears the drawing
  • 'F5' restarts the program

Programming:

  • `turtle` is a global variable for the first turtle
  • `turtle.left()`, .right(), .up(), and .down() move the turtle around
  • `turtle.setColor(htmlcolor)` will change the color of the pen
  • `turtle.x`, .y, .drawing, .facing, .thickness, .color, .isDrawing
  • press `F12` to open the javascript console. Here log messages appear and you can command the turtle to do things
